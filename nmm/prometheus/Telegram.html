How to Create and Connect a Telegram Chatbot

A chatbot is an automated multifunctional assistant, that can receive send and send triggered messages, and with SendPulse, your bot can save information as variables for future usage.

Follow the step-by-step instruction to create your first chatbot for Telegram messenger or connect an existing one to SendPulse for further configuration.

If you already have a bot, skip the first paragraph and take a look at the next section.

How to Create a New Bot for Telegram

Open Telegram messenger, sign in to your account or create a new one.

Step 1. Enter @Botfather in the search tab and choose this bot.

Official Telegram bots have a blue checkmark beside their name.

Click “Start” to activate BotFather bot.

In response, you receive a list of commands to manage bots.

Step 2. Choose or type the /newbot command and send it.

Step 3. Choose a name for your bot — your subscribers will see it in the conversation. And choose a username for your bot — the bot can be found by its username in searches. The username must be unique and end with the word “bot.”

After you choose a suitable name for your bot — the bot is created. You will receive a message with a link to your bot t.me/<bot_username>, recommendations to set up a profile picture, description, and a list of commands to manage your new bot.

To connect a bot to SendPulse you need a token. Copy your token value and find more information about connecting your bot to SendPulse in the last section of this article.

How to Find a Token for an Existing Bot

Step 1. Go to the @BotFather bot and send the command /token.

You will see buttons with any bots that you have created.

Step 2. Choose the one you need a token for so you can connect it with Sendpulse.

Copy the token value.

Create a Channel and retrieve the channel's chat ID

  • Create a channel in telegram and name it whatever you like. e.g. MyINNOG6Monitoring or similar
  • Invite @BotFather to that channel as admin
  • Type at least one message, this is very important
  • Get the Chat id:

Config Telegram in Grafana

  • Login to grafana
  • Click to the left Bell icon
  • Click on Contact points
  • Click on New Contact point
  • Set name as Telegram
  • Select Telegram on Contact point type
  • Put Telegram API token to BOT API token
  • Add chat ID (it starts with -, and that needs too)
  • Click Test button to test telegram notification
  • Click Save contact point button.

Add alert

  • Go to dashboard
  • Click to the left Bell icon
  • Click on Alert rules
  • Click on New alert rule
  • Now click on Code button
  • Now on text box with Enter a PromQL query... type
node_filesystem_avail_bytes{instance="srvX.lab.shakya.io:9100"} / node_filesystem_size_bytes{instance="srvX.lab.shakya.io:9100"} * 100
  • on B section, Function set to Mean
  • on C section, click on Threshold button and change it to Math
  • on Expression, type
$B < 90
  • click preview button

  • now on Add details for your alert section
  • set rule name as free space
  • on folder, click + Add New, then type as general_alert and press enter
  • group as node
  • on summary Freespace left on the instance is below 90%.
  • on description Only {{ humanize $values.B.Value }} % freespace is left on node {{ $labels.instance }}.

  • finally click Save button at top

Change Notification policy

  • Go to dashboard
  • Click to the left Bell icon
  • Click on Notification policies
  • Click Edit button on Root policy - default for all alerts
  • then on Default contact point select Telegram
  • Click save

Now you should start getting alerts on your Telegram Channel
